# ECB (Electronic Code Book Mode)
> 不安全，不要使用

- 將明文按分組大小切分，然後用同樣的金鑰（金鑰不變）正常加密切分好的明文分組。
- 短資料（如加密金鑰）的加密
- 無法隱藏原明文資料（同樣的明文分組加密得到的密文也是一樣的）
- 無加密擴散能力

## 缺點
