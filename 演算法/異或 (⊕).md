# 異或 (Exclusive OR, XOR, ⊕)
>跟 JavaScript 的 Boolean 相反

```
a⊕b = (¬a ∧ b) ∨ (a ∧¬b)
```

如果a、b兩個值
- 不相同，異或結果為1
-  相同，異或結果為0

異或也叫半加運算，其運算法則相當於不帶進位的二進制加法

## 可當加法算（但不要進位）
```
　　1
⊕　0  
-------
　　1
```
`a⊕b = 結果`
```
　　0101
⊕　1110 
----------
　　1011

```

把結果回推也是用異或（`結果⊕b = a`）
```
　　1011
⊕　1110 
----------
　　0101

```
