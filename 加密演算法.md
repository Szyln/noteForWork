# 金鑰管理
- 金鑰管理(key management) 是重要的密碼學工作
- 如果環境允許，盡可能將金鑰及演算法儲存在==硬體安全模組 HSM(Hardware Security Module)== 中
- 確保有權限的使用者可以使用HSM中的加解密演算法
- 密碼或工作金鑰(Working key)定期更換
